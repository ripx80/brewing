# Brewing

## Prozess - Step by Step

1. Rezept
2. Einkaufen
3. Maischen
4. Läutern (Malz/Teber, Feststoffe trennen)
5. Kochen (Hopfen)
6. Whirpool
7. Kühlen (Kühlspirale o. Plattenwärmetauscher)
8. Gären (kontrolliert und temperiert)
